# Sorting Algorithm Benchmark

## Ziele

1. Größe eingeben
2. Zufällig füllen (auf knopfdruck)
3. Start knopf
4. Alle Sortierverfahren nacheinander, und zeit messen

## Implementierte Algorithmen

- [x] bubblesort
- [x] selection-sort
- [x] insertion-sort
- [ ] heapsort
- [ ] mergesort
- [ ] quicksort
